This is a test task. It uses Docker to quickly set up all necessary pre-requisites for running this on any machine.

### Prerequisites

You'll need to have [docker](https://docs.docker.com/get-docker/) installed in order to run this project.

### Installing

From the project root, run
```
./infra/build.sh
```

## Usage

Once you have the project set up and running, go to `http://localhost/` in your browser.

## Built With

* [Composer](https://getcomposer.org/doc/) - Dependency Management
* [Docker Compose](https://docs.docker.com/compose/) - Containerisation
